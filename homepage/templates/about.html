TicFac Store
Un Sistema de Gestión de Inventario


| Acerca de |


TicFac Store es un software sencillo de gestión de inventario creado para cubrir los requerimientos de una empresa. El software es capaz de llevar el control de las entradas y salidas de stock con una factura generada dinámicamente. Cada proveedor cuenta con un perfil donde puede ver el historial de sus transacciones con la empresa.


| Características |


  • Gráfico interactivo para mostrar el stock disponible actual
  • Se mantienen detalles del stock como el inventario disponible
  • Las transacciones de entrada se realizan a través de compras
  • Las transacciones de salida se realizan a través de ventas
  • Cada transacción realizada mediante el software queda registrada
  • La factura se genera dinámicamente y puede ser impresa
  • Se mantienen perfiles separados para los proveedores
